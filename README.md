# 🛡️ SafeHere – Women Safety Web Application

🔗 Live Demo: https://women-safety-app-eight.vercel.app  

---

## 📌 Project Overview

SafeHere is a real-time women safety web application developed to enhance personal security using AI-based risk analysis, emergency alerts, and live location tracking.

The system provides instant response mechanisms during unsafe situations through smart triggers and automated emergency features.

---

## 🎯 Problem Statement

Women safety is a major concern in many regions.  
During emergencies, quick communication and location sharing are critical.

This project aims to provide:
- Instant SOS alerts
- Real-time location tracking
- Automated risk detection
- Quick escape mechanisms

---

## 🚀 Key Features

- 🧠 AI Threat Analysis (movement-based detection)
- 📍 Real-Time Location Tracking
- 🚨 Emergency SOS Button
- 📩 Auto SMS with Live Location
- 📞 Fake Call Escape Feature
- 📸 Silent Camera Capture
- 🔊 Emergency Alarm
- 🎙️ Voice Command Activation
- 🔐 JWT Authentication
- 📊 Risk History Monitoring

## 🏗️ System Architecture

User → React Frontend → Express Backend → MongoDB Database

- Frontend handles UI, AI logic, and tracking
- Backend manages authentication and data storage
- Database stores users, contacts, and stories
